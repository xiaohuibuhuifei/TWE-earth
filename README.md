# TWE-earth

(The Wandering Earth)A product based on cesiumJS without hacking the cesiumJS source code
基于 cesiumJS 但不入侵 cesiumJS 源码

## 项目结构

```
/
├── build/
│   └── rollup.config.js rollup配置
├── src/
│   ├── constant/ 常量
│   ├── core/ 核心功能
│   ├── entity/ 实体相关
│   ├── inspect/ 检测
│   ├── .babelrc.json babel配置文件
│   └── twearth.js 入口文件
├── .gitignore
├── LICENSE 开源协议
└── package.json
```
